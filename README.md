## docker

* build

```bash
docker-compose build --build-arg UID=$UID
```

* create

```bash
docker-compose up -d
```

* start

```bash
docker-compose start
```

* stop

```bash
docker-compose stop
```

* exec 

```bash
docker-compose exec php bash
```
